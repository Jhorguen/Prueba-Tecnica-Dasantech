# Portafolio

Este proyecto consiste en una página web personal con un sistema de login y una sección de dashboard, utilizando HTML, CSS y JavaScript puro.

## Estructura del Proyecto

- `index.html`: Página principal con presentación personal y enlaces a redes sociales.
- `login.html`: Página de acceso para usuarios.
- `dashboard.html`: Panel principal tras iniciar sesión.
- `css/style.css`: Estilos personalizados para toda la web.
- `js/main.js`: Lógica principal en JavaScript.

## Características

- **Diseño responsivo** y moderno.
- Sección de presentación con foto y enlaces a GitHub y LinkedIn.
- Página de login para autenticación.
- Dashboard para usuarios autenticados.
- Uso de herramientas de IA (ChatGPT) para estructuración y soporte técnico.

## Cómo usar

1. Clona este repositorio:
   ```bash
   git clone https://github.com/TU_USUARIO/Prueba-Tecnica-Dasantech.git
   ```
2. Abre el archivo `index.html` en tu navegador.

## Herramientas utilizadas

- HTML5, CSS3, JavaScript
- [Font Awesome](https://fontawesome.com/) para iconos
- [ChatGPT](https://chat.openai.com/) para soporte y estructuración
- [imgBB](https://es.imgbb.com//) para alojamiento de imagenes
